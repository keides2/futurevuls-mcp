## ğŸ‰ What's New

### Flexible `groups.json` Configuration File Placement

The `groups.json` configuration file can now be placed in multiple locations with automatic detection (in priority order):

1. **Current Directory** - The directory where the command is executed (most convenient for users)
2. **Script Directory** - The directory where `futurevuls-mcp.js` is located (npm global installation)
3. **Home Directory** - `C:\Users\[USERNAME]\groups.json` (Windows) or `~/groups.json` (Unix-like systems)

### Key Benefits

- âœ… **User-Friendly**: No need to locate npm's global installation directory
- âœ… **Flexible**: Place the config file wherever it's most convenient
- âœ… **Better Error Messages**: Shows all searched locations when file is not found
- âœ… **Backward Compatible**: Existing installations continue to work without changes
